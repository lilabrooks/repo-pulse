# ADRs

- [0001 Python web stack](0001-python-web-stack.md): FastAPI + uvicorn + httpx + pytest, static no-build frontend. Status: accepted.
- [0002 In-memory TTL cache](0002-in-memory-ttl-cache.md): 5-minute per-repo summary cache plus optional GITHUB_TOKEN from env. Status: accepted.
- [0003 Homepage-to-repo resolution](0003-homepage-to-repo-resolution.md): proposed scope change to resolve non-GitHub URLs to a repo; needs a GOAL.md constraint change. Status: proposed, not implemented.
- [0004 GitHub Actions CI](0004-github-actions-ci.md): Run the offline test, compile, and OKF mapping checks in GitHub Actions. Status: accepted.
